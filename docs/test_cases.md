**Document type:** Manual-first test case table for current prototype

## Notes

- The current WiseWorkout prototype is tested mainly through manual testing.
- Only features found in the repository are included below.
- For all test cases:
  - **Actual Result:** `To be tested`
  - **Status:** `Pending`
- `Manual / Integration` is used only when the case includes Firebase data save or retrieval.
- Advanced automated testing is not the focus here. Where relevant, the Notes column marks `Future automated test candidate`.
